# Product Requirements Document: Todo List REST API

## Описание продукта
REST API для управления персональными задачами (todo list), предоставляющий возможность регистрации пользователей и выполнения CRUD-операций над их задачами. Реализовано на Python FastAPI с хранением данных в PostgreSQL.

## Ключевые функции
1. Регистрация пользователя (email, пароль).
2. Аутентификация пользователя (JWT-токен).
3. Создание задачи (название, описание опционально, статус "не выполнено" по умолчанию).
4. Получение списка задач текущего пользователя.
5. Получение одной задачи по ID.
6. Отметка задачи как выполненной (PATCH /tasks/{id} с изменением поля completed).
7. Удаление задачи.
8. Валидация данных (название не пустое, email корректный и уникальный).

## Технические ограничения
- Язык: Python 3.10+
- Фреймворк: FastAPI
- База данных: PostgreSQL (через SQLAlchemy или asyncpg)
- Аутентификация: JWT (Access token, возможно refresh)
- HTTP API, JSON
- Документация: Swagger (авто генерация FastAPI)

## Критерии успеха
- Все эндпоинты отвечают корректными HTTP статусами (2xx, 4xx, 5xx).
- Время ответа не более 500ms при 1000 запросов.
- Покрытие тестами не менее 80% (юнит, интеграционные).
- Отсутствие уязвимостей: инъекции SQL (через ORM), защита паролей (bcrypt).
